# Replace Generic Svg Social Link Icon With Discord

### Version 0.1d1

#### SS Version v7.1

---

## Quick Install

* Add the following to Settings > Advanced > Code Injection > HEADER.
  
  ```html
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
  ```
  
* Add code from file
  **[replace generic svg social link icon with discord.html](replace%20generic%20svg%20social%20link%20icon%20with%20discord.html#L1)**
  to Settings > Advanced > Code Injection > FOOTER.
