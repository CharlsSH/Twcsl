# Site Wide Image Borders

### Version 0.5.2

#### SS Version 7.1

#### Fluid Engine Compatible : Unknown

## Quick Install

Add code from file
**[site wide image borders.less](site%20wide%20image%20borders.less#L1)**
to Design > Custom CSS.
