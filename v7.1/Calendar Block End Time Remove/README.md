# Calendar Block End Time Remove

### Version 0.1d0

#### SS Version 7.1

---

## Quick Install

* Add code from file
  **[calendar block end time remove.html](calendar%20block%20end%20time%20remove.html#L1)**
  to Page Settings > Advanced > Page Header Code Injection for the page.
